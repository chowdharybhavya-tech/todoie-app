# 🪄 Todoie App – Hogwarts Inspired Todo List

A simple and visually appealing **Harry Potter-inspired Todo application** built using **HTML, CSS, and JavaScript**. The application allows users to manage daily tasks in a magical Hogwarts-themed interface.

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/chowdharybhavya-tech/todoie-app.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

---

## 🎯 How It Works

- View the default todo list.
- Click the **Add Todo** button.
- Enter a new task.
- The task is instantly added to the list.
- Mark tasks as completed using the checkbox.
